## 1. Goal

Port the **DMC (Destination Management Company) template** into the Vite app. After this PR, a sales user can pick the `dmc` template from the Template Selector and get the full DMC experience that exists in legacy `index.html`:

- **2-tab shell**: `üìä B·∫£ng chi ph√≠ Breakdown` + `üïê L·ªãch s·ª≠ Breakdown` only (no Summary / Dashboard / Quote / Payment / Contract / Customer / NCC tabs).
- **6 DMC categories** instead of 12/13: `hotel`, `transport`, `meal`, `sight`, `staff`, `service_fee`.
- **Output-currency selector** (10 currencies) re-renders all totals in the chosen currency without touching underlying VND data.
- **Profit Margin & Service Charge** row at the bottom of breakdown ‚Äî percent of total cost or fixed amount in the output currency.
- **DMC Compare Panel** ‚Äî input DMC package prices per pax for group sizes 20/25/30/35/40; see margin vs breakdown.
- **Separate cloud quote history** at Firestore `viettours/dmc_quote_history` + `dmc_quote_projects/{cloudId}`. Quote-code prefix `DMC.`.
- **DMC hero price cards** in the toolbar: `T·ªïng breakdown` + `Per pax` (output currency).

**Out of scope:**
- PDF/Word/Excel exports for DMC ‚Äî deferred to a future PR (Phase 4 DMC exports).
- Tests ‚Äî refactor convention from PR-3.6.

## 2. Decisions

| Decision | Choice | Rationale |
|---|---|---|
| DMC state location | **Extend `QuoteDraft`** with `outputCurrency`, `dmcPrices`, `dmcMargin` (all optional) | Single source of truth; mirrors legacy state model; persisted via existing `persist` middleware automatically. |
| DMC defaults on new draft | `outputCurrency:'USD'`, `dmcPrices:{20:0,25:0,30:0,35:0,40:0}`, `dmcMargin:{type:'percent',value:0}` | Matches legacy `useState` initial values at `public/legacy.html:8390-8391`. |
| Tabs allowed in DMC mode | Narrow `view` to `'cost' \| 'history'` at the store level | TABS already template-branched in legacy line 8524; clamping at store prevents a stale `view:'summary'` from rendering after switching to DMC. |
| Currency utility location | New `src/lib/currency.ts` exporting `DMC_CURRENCIES`, `toOutputCurrency`, `fmtCurrency`, and `fmtOutput` (rate-guard) | Pure helpers; reused by `CostView`, `QuoteToolbar`, `DMCComparePanel`. |
| `CurrencySelector` | New `src/components/quote/CurrencySelector.tsx` (MUI Menu/IconButton) | Self-contained popover, purple `#8e44ad` accent matching legacy. |
| `DMCComparePanel` | New `src/components/quote/DMCComparePanel.tsx` (presentational, props-driven) | Pure UI, no store coupling. |
| Profit-margin row | Inline in `CostView.tsx` under `template === 'dmc'` guard | One block, one place that knows about `totalCost`; not worth extracting yet. |
| Bad-rate display | Render `‚Äî` via shared `fmtOutput(vnd, cur, rates)` helper | Avoid silently displaying VND as another currency in saved quotes. |
| DMC Firestore docs | `viettours/dmc_quote_history` + `dmc_quote_projects/{cloudId}` (verbatim from legacy `index.html:196-197`) | Existing data lives there; must read/write the same docs. |
| DMC fb fns factoring | Extract `makeQuoteHistoryApi(historyDoc, projectDocFn)` factory; call twice (regular + DMC) | The 500-slice + 20-version cap + undefined-stripping + project-doc cleanup are easy to get subtly wrong; one factory eliminates drift. |
| History store | **Extend `quoteHistoryStore`** ‚Äî hold `quotes` and `dmcQuotes` slices, `visibleQuotes(template?)` selector | Single store, two subscriptions; cleaner than a second store. |
| Quote-code prefix | Reuse existing `generateQuoteCode` (already handles `dmc ‚Üí DMC.`) | No change needed. |
| Compare-panel total | `totalCostVND + marginVND` is what gets per-paxed in the compare table | Matches legacy `index.html:3847`. |
| `QuoteHistoryView` template source | Reads `template` from `useQuoteStore` directly | Already reads other draft fields from store; passing a prop would be ceremony with no decoupling benefit. |
| Permission gates | DMC `history` tab gated by `hasPerm(user,'viewHistory')` ‚Äî same as legacy `index.html:8525-8527` | Reuse `PERMISSIONS`. |
| HistPanel sidebar in DMC | Hidden | Legacy DMC has no local-history sidebar drawer. |

### Data flow for currency display

- All internal calcs stay in VND (`computeTotals` unchanged).
- Output rendering goes through `fmtOutput(vnd, outputCurrency, rates)`:
  - `outputCurrency==='VND'` ‚Üí `fmtCurrency(vnd, 'VND')` ‚Üí `1.234.567 ‚Ç´` (vi-VN).
  - `outputCurrency!=='VND'` and `rates[cur]` present ‚Üí `fmtCurrency(vnd / rates[cur], cur)`.
  - `outputCurrency!=='VND'` and `rates[cur]` missing ‚Üí literal `'‚Äî'`.

### `src/lib/currency.ts` (new)

```ts
import type { OutputCurrency } from '@/types';

// Source: public/legacy.html:3667
export const DMC_CURRENCIES: OutputCurrency[] =
  ['USD','VND','EUR','JPY','SGD','KRW','THB','GBP','AUD','CNY'];

export const CURRENCY_FLAGS: Record<OutputCurrency, string> = {
  VND:'üáªüá≥', USD:'üá∫üá∏', EUR:'üá™üá∫', JPY:'üáØüáµ', KRW:'üá∞üá∑',
  SGD:'üá∏üá¨', THB:'üáπüá≠', GBP:'üá¨üáß', AUD:'üá¶üá∫', CNY:'üá®üá≥',
};

// vnd √∑ rate; identity for VND or when the rate is missing.
// Source: public/legacy.html:3694-3697
export function toOutputCurrency(
  vnd: number,
  cur: OutputCurrency,
  rates: Record<string, number>,
): number {
  if (cur === 'VND' || !rates[cur]) return vnd;
  return vnd / rates[cur];
}

// Legacy format rules (public/legacy.html:3698-3702):
//   VND      ‚Üí "1.234.567 ‚Ç´"     (vi-VN locale)
//   JPY/KRW  ‚Üí "1,234,567 JPY"   (no decimals, en-US grouping)
//   else     ‚Üí "1,234.56 USD"    (2 decimals, US grouping via regex)
export function fmtCurrency(amount: number, cur: OutputCurrency): string {
  if (cur === 'VND') return Math.round(amount).toLocaleString('vi-VN') + ' ‚Ç´';
  if (cur === 'JPY' || cur === 'KRW') return Math.round(amount).toLocaleString('en-US') + ' ' + cur;
  return amount.toFixed(2).replace(/\B(?=(\d{3})+(?!\d))/g, ',') + ' ' + cur;
}

// Single rate-guarded display helper used by every DMC render site.
// Returns '‚Äî' when the output currency has no rate set, instead of silently
// rendering VND as USD/EUR/etc.
export function fmtOutput(
  vnd: number,
  cur: OutputCurrency,
  rates: Record<string, number>,
): string {
  if (cur !== 'VND' && !rates[cur]) return '‚Äî';
  return fmtCurrency(toOutputCurrency(vnd, cur, rates), cur);
}
```

### `quoteStore.ts` ‚Äî DMC defaults & actions

```ts
function dmcDefaults(): Pick<QuoteDraft, 'outputCurrency' | 'dmcPrices' | 'dmcMargin'> {
  return {
    outputCurrency: 'USD',
    dmcPrices: { 20: 0, 25: 0, 30: 0, 35: 0, 40: 0 },
    dmcMargin: { type: 'percent', value: 0 },
  };

newDraft: (template) => set(() => ({
  draft: {
    ...EMPTY_DRAFT,
    template,
    catEnabled: Object.fromEntries(
      CATS.map(c => [
        c.id,
        template === 'dmc' ? DMC_CAT_IDS.includes(c.id) : c.id !== 'dmc',
      ]),
    ) as Record<CategoryId, boolean>,
    items: seedFactory(template)(EMPTY_DRAFT.pax),
    ...(template === 'dmc' ? dmcDefaults() : {}),
  },
  view: 'cost',
})),

setOutputCurrency: (cur) =>
  set(s => ({ draft: { ...s.draft, outputCurrency: cur } })),

setDmcPrice: (gs, v) =>
  set(s => ({
    draft: {
      ...s.draft,
      dmcPrices: { ...(s.draft.dmcPrices ?? {}), [gs]: v },
    },
  })),

setDmcMargin: (patch) =>
  set(s => ({
    draft: {
      ...s.draft,
      dmcMargin: { ...(s.draft.dmcMargin ?? { type: 'percent', value: 0 }), ...patch },
    },
  })),

setView: (v) => set(s => {
  let next = v;
  if (s.draft.template === 'dmc' && next !== 'cost' && next !== 'history') next = 'cost';
  return { view: next };
}),
```

## 7. Edge cases & gotchas

- **Stale view state.** Persisted draft has `view:'summary'`, then template flips to DMC ‚Üí `setView` guard catches it; `init()` hydration also clamps the persisted view.
- **Missing rate.** `rates[outputCurrency]` is `undefined` while editing the rates panel. `fmtOutput` returns `'‚Äî'` ‚Äî no silent VND-as-USD display in saved quotes.
- **Persist hydration of new optional fields.** Pre-existing per-user `vte_quote_draft_{u}` payloads lack `outputCurrency`/`dmcPrices`/`dmcMargin`. If `storedDraft.template === 'dmc'`, `init()` merges `dmcDefaults()` over any missing keys.
- **Firestore `undefined`-rejection.** The factory preserves the `Object.fromEntries(... filter(v !== undefined))` dance from `fbSaveQuote`. Must apply to both regular and DMC paths.
- **`generateQuoteCode` collision space.** Regular and DMC live in *separate* docs. `fbSaveDMCQuote` reads `DMC_QUOTE_HISTORY_DOC` first, so the seq counter is per-day-per-prefix-per-doc ‚Äî no collision with regular `DMC.` quotes (which don't exist anyway in the regular doc).
- **Template switch on existing draft.** Today there is no "convert draft from intl‚Üídmc" flow ‚Äî template is set only by `newDraft`, which resets the draft. No migration path needed.
- **`activeCATS`.** `getCATS('dmc')` already returns the 6 DMC categories ‚Äî no change to `constants.ts`.
