# PR-Menu Implementation Plan

**Goal:** Port the legacy Menu library (Thư viện thực đơn) into the Vite app. Includes the Restaurant library, the Menu builder, DOCX + PDF exports, and the TemplateSelector wiring.

**Spec:** `docs/superpowers/specs/2026-06-09-pr-menu-design.md`

**Tech:** TS · React 18 · MUI v6 · Zustand · Firebase · Sortable.js · docx · jspdf.

**Tests:** Out of scope. Run `typecheck + lint + build` after each task.

**Conventions per task:** Commit per task. Conventional Commits with `feat(menu): …` (or `feat(exports/firebase): …`). Co-author: `Co-Authored-By: Claude Opus 4.7 (1M context) <noreply@anthropic.com>`.
